# Handwritten Digit Recognition Engine using PyTorch

An end-to-end Computer Vision project that trains a Convolutional Neural Network (CNN) to recognize handwritten digits (0-9) using the MNIST dataset.

## 📊 Project Results
* **Final Test Accuracy:** 99.14% on 10,000 unseen images.
* **Architecture:** 2 Convolutional Layers (Conv2d) + Max Pooling + 2 Fully Connected Layers.
* **Optimization:** Adam Optimizer with Cross-Entropy Loss.
